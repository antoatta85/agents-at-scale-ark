# Deploy to Kubernetes

This tutorial walks you through deploying Ark resources to a production Kubernetes cluster.

## What You'll Learn

- How to prepare your cluster
- How to deploy Ark resources
- How to configure networking and access
- How to verify deployment

## Prerequisites

- A Kubernetes cluster (local or remote)
- kubectl configured
- Helm installed
- Understanding of Kubernetes basics

## Step 1: Prepare Your Cluster

Ensure your cluster meets Ark's requirements.

## Step 2: Install Ark

Deploy Ark controller and services to your cluster.

## Step 3: Deploy Your Resources

Apply your agents, teams, and other resources.

## Step 4: Configure Access

Set up networking, ingress, and authentication.

## Step 5: Verify Deployment

Check that all resources are running correctly.

## Next Steps

- Learn about production best practices
- Set up monitoring and observability
- Configure backup and restore
