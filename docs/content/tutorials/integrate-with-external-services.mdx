# Integrate with External Services

This tutorial shows you how to integrate agents with external services and APIs.

## What You'll Learn

- How to create tools for external APIs
- How to handle authentication
- How to manage API rate limits
- How to test integrations

## Prerequisites

- Ark installed and running
- Access to external APIs you want to integrate
- Understanding of REST APIs

## Step 1: Design Your Tool

Plan how your agent will interact with the external service.

## Step 2: Create Tool Resource

Define the tool using Ark's Tool CRD.

## Step 3: Configure Authentication

Set up API keys, OAuth, or other authentication methods.

## Step 4: Attach to Agents

Connect your tool to agents that need it.

## Step 5: Test Integration

Execute queries that use the external service and verify responses.

## Next Steps

- Learn about tool error handling
- Explore advanced tool patterns
- Set up monitoring for external services
