# Add Custom Tools via MCP

This tutorial shows you how to add custom tools to your agents using the Model Context Protocol (MCP).

## What You'll Learn

- What MCP is and why it's useful
- How to set up an MCP server
- How to connect MCP servers to agents
- How to test MCP tool integration

## Prerequisites

- Ark installed and running
- Basic understanding of tools and agents
- Docker installed (for MCP server deployment)

## Step 1: Understand MCP

Model Context Protocol enables agents to access external tools and services.

## Step 2: Deploy an MCP Server

Deploy a filesystem MCP server or create your own.

## Step 3: Create MCPServer Resource

Configure the MCPServer CRD to connect to your MCP server.

## Step 4: Attach Tools to Agents

Reference MCP tools in your agent configurations.

## Step 5: Test Tool Execution

Execute queries that use MCP tools and verify functionality.
