# Agents at Scale Marketplace

> **üìç Documentation Location**: Complete marketplace documentation is available in a separate documentation site.

The ARK Marketplace provides a curated collection of services, agents, and tools that extend the capabilities of the Agents at Scale platform.

## View Marketplace Documentation

To access the full marketplace documentation:

```bash
cd marketplace/docs
npm install
npm run dev
# Visit http://localhost:3000
```

The marketplace documentation site includes:
- **Services**: Detailed installation and configuration guides
- **Integration Examples**: How to integrate services with ARK
- **Development Guides**: Local development workflows
- **Contributing Guidelines**: How to add new services

## What is the Marketplace?

The marketplace is located at `marketplace/` in the repository root and includes:

- **Services**: Production-ready observability, storage, and infrastructure services
- **Agents**: Pre-built agents and agent templates (coming soon)
- **Tools**: Reusable tools and utilities for agent development (coming soon)
- **Projects**: Complete solution accelerators and reference implementations (coming soon)

## Available Services

Currently available services:

- **[Phoenix](/marketplace/services/phoenix)** - AI/ML observability and evaluation platform with OpenTelemetry integration

## Quick Install Example

```bash
# Example: Install Phoenix
cd marketplace/services/phoenix
devspace dev  # Deploy + dashboard
```
