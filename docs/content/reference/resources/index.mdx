# Resources

Complete reference documentation for all ARK Custom Resource Definitions (CRDs) and resource types.

## Available Resources

- **[Agents](./agent)** - Autonomous AI agents with specific capabilities and tools
- **[Teams](./team)** - Multi-agent coordination and orchestration
- **[Models](./models)** - AI model configurations and providers
- **[Queries](./query)** - Query execution and conversation management
- **[Tools](./tools)** - Custom tools and MCP tool references
- **[MCPServers](./mcpserver)** - Model Context Protocol server configurations
- **[Memories](./memory)** - Persistent storage for agent conversations and state
- **[A2AServers](./a2aserver)** - Agent-to-agent communication services

## Resource Relationships

Understanding how resources relate to each other is crucial for designing effective agentic workflows. See [Resource Relationships](/reference/relationships) for detailed information on resource dependencies and common patterns.

## Resource Specifications

Each resource type has detailed specifications including:
- Complete YAML schema
- Field descriptions and validation rules
- Usage examples
- Best practices
- Common patterns
