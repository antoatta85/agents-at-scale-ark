# Contributors

This project follows the [All Contributors](https://github.com/all-contributors/all-contributors) specification to recognize contributions of all kinds.

<ContributorsGrid />

## How to Add Contributors

Comment on any issue or PR with:

```
@all-contributors please add @username for code,doc,ideas
```

Available contribution types: `code`, `doc`, `ideas`, `bug`, `review`, `test`, `maintenance`, `projectManagement`
