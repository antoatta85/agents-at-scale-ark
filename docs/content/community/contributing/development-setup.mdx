# Development Setup

This guide helps you set up a local development environment for contributing to Ark.

## Prerequisites

- Kubernetes cluster (minikube, kind, or Docker Desktop)
- kubectl configured
- Docker installed
- Go installed (for controller development)
- Node.js installed (for dashboard development)
- Python installed (for API development)

## Local Development

See the [Local Development Guide](../../how-to-guides/development/local-development-setup) for detailed instructions on setting up local development with DevSpace.

## Quick Start

```bash
# Install DevSpace
brew install devspace uv

# Start development environment
devspace dev
```

## Development Workflow

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test locally
5. Submit a pull request

## Testing

- Run unit tests
- Run integration tests
- Test your changes locally
- Verify documentation
