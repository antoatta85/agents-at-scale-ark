# Support Channels

Multiple channels are available for getting help with Ark.

## GitHub

### Issues

Report bugs and request features:
- [GitHub Issues](https://github.com/mckinsey/agents-at-scale-ark/issues)
- Search existing issues first
- Provide detailed information

### Discussions

Ask questions and share ideas:
- [GitHub Discussions](https://github.com/mckinsey/agents-at-scale-ark/discussions)
- Community Q&A
- Feature discussions

- Steps to reproduce
- Relevant logs
