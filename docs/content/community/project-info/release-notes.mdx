# Release Notes

Release notes for Ark versions are available on [GitHub Releases](https://github.com/mckinsey/agents-at-scale-ark/releases).

## Version Format

Ark follows semantic versioning:
- **Major**: Breaking changes
- **Minor**: New features, backward compatible
- **Patch**: Bug fixes, backward compatible

## Release Information

Each release includes:
- New features
- Bug fixes
- Breaking changes
- Migration guides
- Upgrade instructions

## Upgrade Guide

When upgrading:
1. Review release notes
2. Check for breaking changes
3. Review migration guides
4. Test in non-production first
5. Follow upgrade instructions

See the [Upgrading Guide](../../reference/upgrading) for detailed upgrade instructions.
