# Migration Guides

Migration guides help you upgrade Ark and migrate your resources between versions.

## Upgrade Process

1. Review release notes for breaking changes
2. Backup your resources
3. Follow version-specific migration guide
4. Test in non-production environment
5. Apply to production

## Version-Specific Guides

Migration guides are included in release notes for versions with breaking changes.

## Common Migrations

### Resource Format Changes

Some versions update resource formats:
- Review new format
- Update resource definitions
- Test resources before applying

### API Changes

API changes may require:
- Updating client code
- Modifying configurations
- Adjusting integrations
